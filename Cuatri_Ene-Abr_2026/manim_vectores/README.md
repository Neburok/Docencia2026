# Manim Vectores Cálculo Multivariable

Visualizaciones educativas con Manim para conceptos fundamentales de vectores.

## Estructura del Proyecto
manim_vectores_calculo/
 ├
── scenes/ # Archivos de escenas Manim ├── config/ # Archivos de configuración ├── output/ # Videos y renderizados │ ├── videos/ │ └── images/ ├── requirements.txt # Dependencias └── README.md # Este archivo


## Videos (10 conceptos fundamentales)

1. **video1_representacion_2d.py** - Representación de vectores en 2D
2. **video2_representacion_3d.py** - Representación de vectores en 3D
3. **video3_magnitud.py** - Magnitud de vectores
4. **video4_direccion.py** - Dirección de vectores
5. **video5_suma.py** - Suma de vectores
6. **video6_resta.py** - Resta de vectores
7. **video7_multiplicacion_escalar.py** - Multiplicación por escalar
8. **video8_producto_punto.py** - Producto punto
9. **video9_producto_cruz.py** - Producto cruz
10. **video10_aplicaciones.py** - Aplicaciones prácticas

## Instalación

```bash
pip install -r requirements.txt
Ejecutar escenas
# Renderizar una escena específica
manim -pql scenes/video1_representacion_2d.py VideoRepresentacion2D

# -p: preview (abrir el video después de renderizar)
# -q: calidad (l=low, m=medium, h=high, k=4k)
Estilo Minimalista
Fondo negro
Vectores con colores vibrantes
Animaciones suaves
Texto claro y conciso
Duración de videos: 2-3 minutos
```
