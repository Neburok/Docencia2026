# Repositorio de Materiales Docentes

Este repositorio contiene archivos de planeación de clases, guías de estudio, presentaciones para clases presenciales y evaluaciones, alineados al Enfoque Basado en Competencias (EBC) de la Universidad Tecnológica de Querétaro (UTEQ).

## Alcance

- Planeación de clases para cuatrimestre de 15 semanas (13 semanas efectivas + 2 semanas de evaluaciones).
- Guías de estudio y material de apoyo.
- Presentaciones (PowerPoint/PDF) para sesiones presenciales.
- Evaluaciones y rúbricas.

## Estructura del repositorio

- planeacion_clase/
- guias_estudio/
- presentaciones/
- evaluaciones/
- plantillas/
- proyectos_integradores/

Cada carpeta contiene archivos en formatos Markdown (.md), PowerPoint (.pptx), PDF (.pdf) y otros según necesidad. Usa nomenclatura consistente descrita a continuación.

## Convenciones de Nomenclatura

Formato recomendado para archivos Markdown (planeaciones y guías):

- Planeacion_<MateriaCodigo>_<Semestre>_S<SemanaXX>.md
- GuiaEstudio_<MateriaCodigo>_<Semestre>_S<SemanaXX>.md
- Presentacion_<MateriaCodigo>_<Semestre>_S<SemanaXX>.<ext> (pptx o pdf)
- Evaluacion_<MateriaCodigo>_<Semestre>_S<SemanaXX>.<ext>

Ejemplos:

- Planeacion_Fis101_Sem1_S01.md
- GuiaEstudio_Fis101_Sem1_S01.md
- Presentacion_Fis101_Sem1_S01.pptx
- Evaluacion_Fis101_Sem1_S01.pdf

Notas:
- MateriaCodigo es el código asignado en el plan de estudio.
- Semestre puede ser "S1" o "S2" para el cuatrimestre actual.
- "S01" es la semana 1 de las semanas efectivas; ajusta según tu calendario.

Plantillas disponibles en plantillas/:
- Plantilla_Planeacion.md
- Plantilla_GuiaEstudio.md
- Plantilla_Rubrica.md
- Plantilla_Autoevaluacion.md
- Plantilla_Presentacion.md

## Plantillas y Formatos

- Plantilla_Planeacion.md: estructura para objetivos, competencias, recursos, actividades y evaluación.
- Plantilla_GuiaEstudio.md: formato para guías de estudio y ejercicios.
- Plantilla_Rubrica.md: rúbricas de evaluación por criterios.
- Plantilla_Autoevaluacion.md: instrumentos de autoevaluación y coevaluación.
- Plantilla_Presentacion.md: pautas para diapositivas, diseño y notas del ponente.

## Flujo de Trabajo (EBC)

- Planificación: definir competencias y resultados de aprendizaje.
- Preparación: crear planeación y guías, y preparar presentaciones.
- Revisión: revisión por pares y aseguramiento de calidad EBC.
- Publicación: subir a las carpetas correspondientes.
- Actualización: versionar y mantener trazabilidad.

## Cómo Contribuir

- Estructura tus cambios en ramas por recurso o materia, p. ej. feature/planeacion-Fis101-S01.
- Asegúrate de que los archivos sigan el formato de nombres recomendado.
- Escribe mensajes de commit claros: "feat(planificacion): agregar planeación semana 1 Fis101" o "docs(readme): actualizar instrucciones".
- Abre Pull Requests para revisión y aprobación.
